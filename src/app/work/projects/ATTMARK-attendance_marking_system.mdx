## Overview

**ATTMARK** is an intelligent attendance marking system built with **Python, OpenCV, and MySQL**.  
It automates student attendance by using **face recognition technology**, capturing live video feeds, detecting and verifying student faces, and marking attendance electronically.  

The system eliminates manual roll calls, reduces errors, prevents proxy attendance, and saves valuable class time.  

---

## Key Features

- **Face Registration**
  - Capture and store facial data of students in a secure database.  
- **Real-Time Detection & Recognition**
  - Live video feed with accurate face detection and recognition.  
- **Automated Attendance**
  - Marks attendance instantly upon successful recognition.  
- **Admin Access**
  - Faculty can view attendance reports and manage student data easily.  
- **Secure & Accurate**
  - Prevents proxy or fraudulent attendance using unique facial features.  

---

## Technologies Used

- **Frontend:** HTML, CSS, JavaScript, PyCharm for development.  
- **Backend:** Python with OpenCV libraries (Haar Cascade, LBPH) for face detection and recognition.  
- **Database:** MySQL for storing student data and attendance records.  
- **Camera:** Droid Cam app or connected cameras for live video capture.
