## Overview

The **Real-Time Chat Application** is a full-stack MERN project designed to enable seamless instant messaging between users. It integrates authentication, user management, and real-time communication using WebSockets, offering a smooth and responsive chatting experience. The project demonstrates the integration of **React frontend** with a **Node.js + Express backend**, powered by **MongoDB** for data storage and **Socket.io** for live updates.

## Challenges and Learnings

One of the biggest challenges was managing **real-time synchronization** between multiple users while ensuring security. Handling **authentication tokens with Socket.io connections** required careful design. Another challenge was building a responsive, clean UI using Tailwind and DaisyUI without breaking component consistency.  

Through this project, I strengthened my understanding of **WebSockets, authentication flows, and MERN stack integration**.

## Outcome

The application successfully demonstrates **real-time communication** with authentication and scalability in mind. It provides a **production-ready boilerplate** for chat apps or other socket-based applications, making it an excellent showcase of **full-stack MERN development skills**.
